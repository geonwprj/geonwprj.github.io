大奉打更人

第二十五章 單嘢勁L難搞

許七安試吓細步跑，「如履平地」完全冇阻礙，佢即刻將佛子件事拋諸腦後，嗰個靚到爆嘅琉璃菩薩俾監正打傷咗，兩三年都走唔開阿蘭陀。

啲護法金剛，甚至乎其他羅漢，就算對自己有威脅，但係只要識得兜路、行開啲，避開危險，羅漢都冇想像中咁得人驚。

打唔贏，咪走囉。

但係對住琉璃菩薩呢啲擅長速度同埋控制嘅一品高手，想走都走唔甩。

柳芸行得好辛苦咁行緊，當佢行入咗呢條菩薩羅漢企喺兩邊嘅路之後，巨大嘅威壓由天而降，呢股講唔出嘅壓力唔係壓落肉身，而係壓落人哋嘅內心。

每行完一步，就對佛門多一分認同，好似經歷緊一個慢動作嘅洗腦過程咁。

之所以行得咁辛苦，係因為原本嘅諗法同呢股外來嘅理念喺度抗衡緊。

是但凡有智慧有主見嘅生物，對於洗腦都係本能咁抗拒。

咁嘅情況喺佢預料之中，佢身為雷州本地江湖勢力，佢接觸過唔少曾經想過要遁入空門嘅「信徒」，呢啲信徒雖然最後失敗咗，但係由浮屠寶塔出嚟之後，就越嚟越虔誠。

「我可以試吓接受呢種「灌輸」，主動接納呢份認同感，咁樣會唔會令我行快啲呢？」

佢做咗相應嘅嘗試，驚喜咁發現速度真係快咗少少。

由此得出結論，資質好，用心接納佛門理念，係會令速度變快啲，但係最核心嘅係其他嘢，因為佢嘅速度只係快咗少少，冇想像中咁誇張。

至於嗰個核心係乜嘢，柳芸就諗唔明。

呢個時候，佢眼尾睄到有個人影由自己身邊經過。

咁快嘅？

佢呆咗咁望過去。

「我行先喇！」

發覺到佢望住自己嘅許七安，平靜咁點頭，然後，平靜咁行遠咗。

睇住佢行遠嘅身影，柳芸腦海入面得返四個字：閒庭信步。

佢慢慢咁擘大個口，瞪大對眼。

「完全冇影響？佢，佢點可能完全冇影響㗎。就算係佛門嘅和尚，都明顯受到壓制啦，但係佢根本同平時一樣。」

柳芸個腦亂晒籠，諗唔明點解。

就咁樣，許七安追過咗一個又一個雷州本地嘅人，喺佢哋目瞪口呆嘅眼神入面，一騎絕塵。

嗰啲專心致志行緊路嘅人，呆呆咁睇住呢一幕。

「喂，喂，搞乜嘢呀？」

「我哋行嘅唔係同一條路嚟咩，點解佢可以做到咁輕鬆嘅。」

好多人因此停低腳步喺度睇，驚訝咁議論紛紛。

最先聽到身後議論聲嘅，係袁義、李少雲、東方姐妹同埋雙刀門門主湯元武。

佢哋企喺中間位置，聽到身後嘅驚嘆聲同埋議論聲。

東方姐妹疑惑咁擰轉頭望過去，花容失色，視線入面，嗰個青衣人慢步咁行過嚟，冇停頓，輕鬆自在。

「咦？」

扛住長槍嘅李少雲猛咁轉身，槍桿跟住橫掃，身邊嘅都指揮使袁義即刻縮低頭，避開咗槍頭嘅橫掃。

啱啱想鬧呢個下屬，但係跟住佢嘅目光望過去，即刻滿臉錯愕。

「喂，你點做到㗎，可唔可以分享吓經驗呀。」李少雲笑住咁講。

東方姐妹同埋袁義、湯元武即刻望過嚟。

許七安冇停低腳步，冷淡咁應咗句：「天賦呢啲嘢點分享呀。」

李少雲擘大個口，冇聲出。

等到嗰個青衣人行遠咗，佢細聲講：「奶奶哋，呢條友真係天生做和尚嘅材料嚟喎。」

你老味先係做和尚嘅材料........許七安嘴角抽搐咗一下，加快腳步。

袁義眯埋對眼，目光一直喺佢對腳度，細聲講：「完全冇停滯，點可能㗎。」

東方婉清皺晒眉頭：「家姐，呢個人周身都古古怪怪。」

東方婉蓉面色嚴肅咁「嗯」咗一聲，傳音講：

「佢會比三花寺嘅和尚更快一步入第二層。但係唔緊要，佛門啲和尚話，第二層一早已經俾師尊嘅力量侵蝕咗，佢會被困喺嗰度。」

「但係都唔可以俾佢咁順利超越我哋。」

端木婉蓉搖頭：

「你仲未發覺到咩，塔入面有戒律㗎，好難郁手，起碼第一層有戒律。浮屠寶塔係供奉舍利子同埋囚禁高手嘅法器嚟㗎。如果咁容易就郁到手，仲點樣囚禁高手呀？」

東方婉清大聲嗌：「淨心大師，望吓你後面。」

後面？前邊啲和尚擰轉頭望過嚟，佢哋對眼慢慢咁瞪大瞪圓，唔敢相信嘅表情凝固喺塊面度。

就算係淨心同恆音呢啲禪師，心入面都覺得好荒謬。

喺佛門菩薩同金剛嘅「注視」之下，一個外人，竟然行得咁輕鬆自在，掉返轉睇佢哋呢啲佛門弟子，就「步步為營」，仲要俾人壓制到死死吓。「施主係乜水呀？」

淨心停低腳步，望住越嚟越近嘅許七安。

班和尚死死咁望住佢。

我係你哋佛門永遠都得唔到嘅男人喇... 許七安繼續行：「大奉武夫。」

雙方就咁啱身而過。

班佛門和尚呆晒咁望住佢背脊。

淨心和尚收返目光，望實手上面粒鏡獸眼淚凝結成嘅珠仔。

度難師叔，應該睇到啱啱嗰幕喇。

... ... 塔出面。

伊爾布把聲喺度迴盪：「度難，呢條友係邊個呀，點解可以喺浮屠寶塔入面嚟去自如嘅？」

盤龍主持托住粒寶珠，塊皺晒皮嘅老臉一臉嚴肅。

周圍嘅溫度突然之間高咗好多，一陣熱浪吹埋嚟，度難金剛嘅身影出現喺盤龍主持側邊，伸手搶過粒寶珠，凝神細望。

慕南梔好奇咁打量住突然出現嘅度難，呢個和尚身高九尺，又高又大隻，後腦着住個唔會熄滅嘅光環。

呢個就係佛門嘅護法金剛嚟㗎？

小白狐縮埋喺佢心口，震晒咁話：「好，好㷫呀，好㷫呀... ... ...」

慕南梔攬實小白狐，即刻退後，退到佢細細隻身體唔再震先停低。

度難金剛一路細望，一路話：

「浮屠寶塔第一層有戒律之力，法寶唔會出問題嘅，咁即係呢位施主有問題啦。可以喺第一層自由自在咁行嘅，得同樣控制到戒律嘅菩薩同羅漢啫。

「就算係我入到去，都會受到影響。」

伊爾布哼一聲話：「你係咪話，呢條友係佛門嘅菩薩或者羅漢呀？」

度難慢慢搖頭：「當年法濟菩薩將浮屠寶塔放喺呢度嗰陣，設咗禁制，四品以上嘅人，入唔到去。羅漢入唔到，菩薩想入去，淨係可以夾硬破開禁制。」

「咁點解釋眼前發生嘅事呀？」

伊爾布問。

度難金剛唔出聲，佢心入面諗到一個猜測：可能係羅漢轉世，同佛門有因果，所以可以唔理啲戒律，直達佛陀金身面前。

伊爾布諗咗陣，話：「算啦，好彩佢都過唔到第二層嘅。」

李靈素喺遠處聽到兩個超凡人物講嘢，戚咗戚牙，徐謙呢條老嘢，究竟係乜嘢人嚟㗎？

又同佛門扯上關係喇？

同司天監關係唔簡單，身上面有咁多蠱術，而家又好似同佛門好有淵源，佢究竟係邊個嚟㗎... ... ...

... ... ... 過咗一陣，許七安順利咁行到佛陀金身面前，抬頭望住高到好似山咁嘅金身，真係好宏偉壯觀。

「浮屠寶塔得三層啫，第一層係用嚟考核人才嘅，唔係好難，幾乎冇乜危險性。咁，第二層或者第三層，可能就係封印神殊同納蘭天祿嘅地方喇。

「我又要搶返啲龍氣，又要解開神殊封印，仲要阻止佢哋放納蘭天祿出嚟，任務都幾重手吓... ... ... ...

「黐喺法寶上面嘅龍氣應該點樣攞返嚟好呢？ 唔通要殺咗件法寶咩。 一品菩薩嘅法寶，點睇都係得俾人反殺嘅下場。」

許七安冇即刻入第二層，望住金身。扮晒發緊夢咁，腦入面諗緊嘢。

佢靜靜雞伸手入心口，揸住地書碎片，口入面唸唸有詞，諗住用監正教佢嘅口訣，用龍氣同國運相吸嘅特性，再加埋地書碎片，吸啲龍氣。

可惜都係唔得。

啲龍氣冇反應，同個寶塔痴纏到死，唔理佢嘅召喚。

「係咪浮屠寶塔級數太高呢？ 佛門都係為咗啲龍氣嚟，我可以喺暗地度睇定啲，坐山觀虎鬥。 反而係解神殊個印同阻止納蘭天祿甩身呢兩件事比較麻煩。

「前者有二師兄教我嘅解印口訣，但係解到監正嘅封印，未必解到浮屠寶塔本身嘅封印。 後者，我除非殺晒東方姐妹仲有班佛門和尚，唔係點阻止到納蘭天祿甩身呀？

「盡人事聽天命啦，攞到龍氣就穩賺唔蝕，神殊件事唔得嘅話第日再算。 至於納蘭天祿，唔可以強求。得我一個人啫，盡力就得㗎喇。 監正真係嘅，俾咁高難度嘅任務我。」

「不如入咗第二層探吓路先，諗定點樣可以坐山觀虎鬥嘅計劃。」

佢即刻為自己定好目標，龍氣一定要攞到，神殊盡力爭取，阻止納蘭天祿甩身就隨緣啦。

見到淨心佢哋一步一步咁行埋嚟，許七安唔再猶豫，向住佛陀金身拜咗三拜。

下一刻，雲霧圍繞嘅天頂，照落嚟一道金光，佢就喺第一層消失咗。

... ... ... 許七安最先感受到嘅係暖笠笠嘅陽光，仲有爛晒嘅大地，呢度好似啱啱打完一場好激嘅仗咁。

呢度係一片好廣闊嘅曠野，天空好藍，天氣又乾又凍。

「呢度係佛境嚟㗎？ 點都冇啲佛境應該有嘅祥和氣息嘅喂.........」佢心諗， 耳邊就聽到把好熟口面， 又好溫柔嘅聲線：

「今日，你死梗。」

佢擰轉頭望過去， 就見到唔遠處企咗個着住青色衫嘅人， 五官清秀俊俏， 身形修長， 對眼清澈明亮， 都仲未有歷盡滄桑嘅感覺。 兩邊鬢角都冇變白。

魏淵！
